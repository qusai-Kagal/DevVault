# 🎲 Dicee Game

A simple dice game that generates random dice rolls at the click of a button!

## ✨ Features

- 🎯 Random dice generation
- 🏆 Winner declaration
- 🔄 Refresh to play again
- 📱 Responsive design

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- DOM manipulation

## 🚀 Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Click to roll the dice!
