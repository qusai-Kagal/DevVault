## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- `pygame` library  
  Install via:
  ```bash
  pip install pygame
  ```

### 🛠️ Run Any Game
Navigate to the desired project folder and run:
```bash
python main.py
```

Example:
```bash
cd [game-folder]
python main.py
```
