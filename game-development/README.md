## 👨‍💻 About This Portfolio

Welcome to DevVault's game development collection - a comprehensive showcase of interactive entertainment projects spanning from classic console games to modern GUI applications. This portfolio demonstrates proficiency in game mechanics design, user interface development, and algorithmic problem-solving through engaging, playable experiences.

## 🚀 Getting Started

### 📋 Prerequisites

```bash
# Required Software
Python 3.6+              # Core programming language
pip                       # Package installer
pygame                    # For graphics-based games
```

### ⚡ Quick Start Guide

```bash
# 1. Clone the repository
git clone https://github.com/qusai-Kagalwala/DevVault.git
cd DevVault/game-development

# 2. Choose your game category
cd aim-trainer              # For graphics-based games

# 3. Install dependencies (if required)
pip install pygame         # For Pygame-based games
pip install windows-curses  # For Windows curses support

# 4. Run the game
python aim_trainer.py

# 5. For web games
cd ../rock-paper-scissors
open index.html            # Open in your browser
```
