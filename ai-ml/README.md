## 🏆 Featured Projects

### 🎙️ **Real-Time AI Translation**

<details>
<summary><strong>VoiceBridge - Live Multi-Language Voice Translator</strong></summary>

**🎯 Problem Solved**: Breaking language barriers with real-time voice translation

**💡 Key Features**:

- Instant translation powered by Google Translate API
- Natural text-to-speech output with high-quality voices

<details>
<summary><strong>Movie Recommender - Content-Based ML Engine</strong></summary>

**🎯 Problem Solved**: Personalized content discovery using machine learning

**💡 Key Features**:

- Feature engineering on genres, cast, crew, keywords, and plot

**🔬 AI/ML Concepts**: Feature engineering, vectorization, cosine similarity, content-based filtering, NLP preprocessing

</details>

### 🧞‍♂️ **Natural Language Processing**

<details>
<summary><strong>QueryGenie - Text-to-SQL AI Converter</strong></summary>

**🎯 Problem Solved**: Democratizing database access through natural language

**💡 Key Features**:
- Converts plain English to SQL queries using local LLMs
- Automatic database schema detection and analysis
- Clean SQL output without explanations or formatting
- Local AI processing using Ollama and DeepSeek-R1
- Interactive web interface with real-time conversion

**🛠 Tech Stack**: `Python` `Streamlit` `Ollama` `DeepSeek-R1` `LangChain` `SQLAlchemy`

**📈 Business Impact**: Enables non-technical users to query databases, reducing dependency on SQL experts

**🔬 AI/ML Concepts**: Large language models, prompt engineering, natural language understanding, code generation

</details>

### 🤖 **Autonomous Navigation**

<details>
<summary><strong>Reeborg Maze Solver - Rule-Based AI Pathfinding</strong></summary>

**🎯 Problem Solved**: Autonomous robot navigation using algorithmic decision-making

**💡 Key Features**:
- Right-hand rule pathfinding algorithm implementation
- Autonomous obstacle avoidance and goal-seeking behavior
- Real-time decision making based on sensor input
- Deterministic navigation with guaranteed path finding
- Educational AI project demonstrating fundamental robotics concepts

**🛠 Tech Stack**: `Python` `Reeborg's World` `Algorithms` `Decision Trees`

**📈 Business Impact**: Foundational concepts used in robotics, autonomous vehicles, and warehouse automation

**🔬 AI/ML Concepts**: Rule-based AI, sensor fusion, autonomous decision making, pathfinding algorithms

</details>

### 🧩 **Algorithm Implementation**

<details>
<summary><strong>BFS Maze Traversal - Graph-Based Shortest Path</strong></summary>

**🎯 Problem Solved**: Optimal pathfinding in 2D grid environments

**💡 Key Features**:
- Breadth-First Search algorithm for shortest path guarantee
- Efficient graph traversal with O(V + E) time complexity
- Visual path exploration and reconstruction
- Memory-optimized queue-based implementation
- Works with any 2D grid-based maze configuration

**🛠 Tech Stack**: `Python` `Data Structures` `Graph Algorithms` `Queue Implementation`

**📈 Business Impact**: Core algorithm used in GPS navigation, network routing, and game AI

**🔬 AI/ML Concepts**: Graph theory, search algorithms, optimization, computational complexity analysis

</details>

## 🚀 Getting Started

### 📋 Prerequisites

```bash
# Required Software
Python (v3.7+)     # Primary programming language
pip/conda          # Package management
Ollama             # Local LLM serving (for QueryGenie)
Microphone         # Audio input (for VoiceBridge)
Internet           # API access for translation services
```

### ⚡ Quick Start Guide

```bash
# 1. Clone the repository
git clone https://github.com/qusai-Kagal/DevVault.git
cd DevVault/ai-ml

# 2. Choose your project (example: VoiceBridge)
cd VoiceBridge

# 3. Install dependencies
pip install -r requirements.txt

# 4. Set up environment (if needed)
cp .env.example .env
# Edit .env with your API keys

# 5. Run the application
python live_lang_translator.py
```

### 🐳 Docker Support

```bash
# Build and run with Docker
docker build -t devvault-ai .
docker run -p 8501:8501 devvault-ai

# Or use docker-compose for multi-service projects
docker-compose up --build
```
